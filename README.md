
# AUV Sonar Object Recognition Research

Underwater acoustic data analysis and model development for autonomous underwater 
vehicle (AUV) object recognition, conducted at the Underwater Robotics & Autonomous 
Systems Group (URASG), CSIR-Central Mechanical Engineering Research Institute, 
Durgapur, India.

## Research Scope

- Sonar signal processing and acoustic feature extraction
- Object classification model development for underwater environments
- Integration of sonar systems with AUV navigation pipeline
- Field data collection, validation, and analysis

## Institution

**CSIR-CMERI (Central Mechanical Engineering Research Institute)**  
Underwater Robotics & Autonomous Systems Group (URASG)  
Ministry of Science & Technology, Govt. of India  
Durgapur, West Bengal — January 2024 to July 2024

## Note on Data & Code
