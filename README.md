# AUV Sonar Object Recognition

Underwater acoustic data analysis and model development for autonomous 
underwater vehicle (AUV) object recognition, conducted during research 
internship at CSIR-CMERI Durgapur, India.

## Note on Data & Code
Research data, processing pipelines, and model implementations are protected 
under CSIR-CMERI intellectual property agreements and cannot be shared publicly. 
This repository documents the research scope and methodology.

## Research Scope
- Sonar signal processing and acoustic feature extraction
- Object classification model development for underwater environments  
- Field data collection and validation
